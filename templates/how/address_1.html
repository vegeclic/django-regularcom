Indiquez vos coordonnées
Vous pouvez indiquez dans cette étape votre adresse postale ainsi que le point relais que vous choisissez si vous avez opté pour la livraison par Mondial Relay.

  1. Coordonnées : renseigner votre adresse postale. Dans votre profil vous pourrez encore ajouter d'autres adresses afin de définir une adresse de livraison, de facturation et/ou de la personne chez qui vous souhaite que le panier soit livré.
  2. Mondial Relay : indiquez les coordonnées du point relais que vous avez choisit.
  3. Choix du point relais : si vous n'avez pas encore déterminé de point relais, cliquez sur ce bouton poun trouver le point relais qui vous convient.
  4. Étape suivante : Validez pour passer à l'étape suivante.
